# Images

Drop your exported screens here using **exactly these file names**. The pages already point to them. Any missing image hides itself gracefully, so you can launch first and add more later.

## Fastest path (4 images + your photo)
In Figma (Festi file) I created a page called **“Export — web images”** with 4 frames already named correctly and set to PNG @2x:

1. Open that page → **Select all** (Cmd/Ctrl + A).
2. In the right panel, under **Export**, click **Export 4 layers**.
3. Drop the 4 PNGs into this `images/` folder. They already have the right names:
   - `work-estimator.png`
   - `work-mlc.png`
   - `work-mortgage.png`
   - `work-youth.png`

These 4 cover **both** the cards on the home page **and** the main screen on each project detail page. (You can delete that Export page in Figma afterward — it's just clones.)

Then add your photo as **`about-photo.jpg`** (the “Tu foto” slot in About). That's everything you need to launch.

> Tip: the exports are 2800px wide (2x). If you want a lighter site, run them through tinypng.com before committing.
