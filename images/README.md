# Images

Drop your exported screens here, using **exactly these file names**. The pages already point to them. Any image that's missing just hides itself gracefully — so you can add them one at a time.

## How to export from Figma
In the **Festi** file, select a frame (or a "Peek"/screen group) → right panel → **Export** → **PNG (2x)** → Export. Rename the file to match the names below and drop it in this folder.
